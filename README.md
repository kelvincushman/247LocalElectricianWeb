# 247LocalElectrician

Your trusted local electrician serving Bilston and the West Midlands. NICEIC approved, fully insured, with 15+ years of experience.

## Services

- **Emergency Callouts** - 24/7 rapid response (30-90 minute arrival)
- **Fault Finding & Repairs** - Professional diagnosis and repairs
- **Fuse Board Upgrades** - Modern consumer unit installations
- **Full & Partial Rewiring** - Complete rewiring solutions
- **Lighting Installation** - Indoor and outdoor lighting
- **Socket Installation** - Additional power points
- **EV Charger Installation** - Home charging solutions
- **EICR Certificates** - Landlord safety inspections

## Service Areas

Covering a 15-mile radius from Bilston including:
- Wolverhampton
- Walsall
- Dudley
- West Bromwich
- Birmingham
- Tipton
- Wednesbury

## Tech Stack

- React 18 + TypeScript
- Vite (with SWC)
- Tailwind CSS
- shadcn/ui components
- React Router DOM

## Development

```bash
# Install dependencies
npm install

# Start development server (port 8080)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```
