## Submission Tips

1. **Use consistent NAP (Name, Address, Phone)** across all directories
2. **Use the same business description** (or variations) everywhere
3. **Add photos** wherever possible - real work photos perform best
4. **Respond to reviews** on all platforms promptly
5. **Keep listings updated** - check quarterly for accuracy
6. **Track which directories send traffic** via Google Analytics UTM parameters
